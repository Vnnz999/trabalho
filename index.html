  • HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar conteúdo na web. Ele define a estrutura básica de uma página web, utilizando tags (ou "marcadores") que informam ao navegador como exibir o conteúdo, como texto, imagens, links, tabelas, formulários, entre outros elementos.
  • CSS (Cascading Style Sheets) é uma linguagem usada para descrever a aparência e o layout de páginas web. Ela permite controlar como os elementos HTML são exibidos, como cores, fontes, espaçamento, posicionamento e outros aspectos visuais. O CSS é fundamental para separar o conteúdo (HTML) da apresentação (estilos), tornando o desenvolvimento web mais organizado e flexível.
